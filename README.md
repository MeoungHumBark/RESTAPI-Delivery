# springAPI-DeliveryApp
## 1. URL

(1) 음식 전체 조회 : GET /foods

(2) 음식 단일 조회 : GET /foods/{id}

(3) 음식 등록 : POST /foods

(4) 음식 수정 : PUT /foods/{id}

(5) 음식 전체 삭제 : DELETE /foods

(6) 음식 개별 삭제 : DELETE /foods/{id}
